# Changelog

## 0.1.0

- Initial public package structure.
- Added shared TRON + EVM WalletConnect auth types.
- Added default `tron:0x2b6653dc` + `eip155` proposal.
- Added browser helpers for one-modal WalletConnect approval.
- Added TRON nonce signing helper.
- Added server Zod schema, TRON signature verification, and namespace validation.
- Added optional Express adapter and memory nonce store.
- Added headless `onUri` mode for custom QR rendering without the default modal.
